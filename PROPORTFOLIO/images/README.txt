HOW TO ADD YOUR IMAGES
======================

1. PROFILE IMAGE:
   - Add your profile photo here
   - Name it: profile.jpg (or profile.png)
   - Recommended size: 600x600 pixels (square)
   - The code is already set to use "images/profile.jpg"

2. PROJECT IMAGES (Optional):
   - Add project screenshots here
   - Name them: project1.jpg, project2.jpg, etc.
   - Recommended size: 800x600 pixels
   - Update the image paths in script.js

SUPPORTED FORMATS:
- JPG / JPEG
- PNG
- WebP

TIP: Keep file sizes under 500KB for faster loading!
